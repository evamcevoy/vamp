# 🧛 Vampire Lore Data Analysis

## 📌 Overview
This project explores a fictional vampire dataset using Python, pandas, and matplotlib.  
It was designed to demonstrate core data analysis skills including data processing, grouping, comparison, and visualisation.

---

## 📊 Dataset
The dataset includes fictional vampire characters and attributes such as:
- Origin region  
- Century  
- Feeding type  
- Weakness  
- Transformation ability  
- Estimated victims per month  
- Survival outcome  

---

## 🛠️ Tools Used
- Python  
- pandas  
- matplotlib  
- Google Colab / Jupyter Notebook  

---

## 🔍 Key Analysis
The analysis focused on:
- Comparing average victims per month across regions  
- Exploring survival rates based on different weaknesses  
- Analysing the relationship between behaviour and outcomes  
- Identifying patterns across different time periods

## 💡 Key Insights
- Certain weaknesses (e.g. sunlight) are associated with lower survival outcomes  
- Some regions show consistently higher average victim counts  
- Higher victim rates do not necessarily correspond to survival in narratives  
- Behavioural traits vary across different historical settings  

---

## 🎯 What This Project Demonstrates
- Data cleaning and preprocessing  
- Exploratory data analysis (EDA)  
- Grouping and aggregation using pandas  
- Interpreting and communicating data-driven insights  
- Creating clear visualisations to support analysis
